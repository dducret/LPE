# 0.2.0 Protocol Depth Gates

## Current State/Functionality Overview

The `0.2.0` protocol depth gates define the minimum release evidence for existing protocol surfaces before adding new protocol breadth. They bind JMAP, IMAP, ActiveSync, EWS, and MAPI over HTTP to canonical `LPE` state and client interoperability checks.

These gates do not move `SMTP` into core `LPE`, do not create protocol-local `Sent`, `Drafts`, or `Outbox` state, and do not widen the documented protocol scope.

## Implementation/Usage

Success criteria:

- each adapter uses canonical mailbox, contact, calendar, task, submission, and rights state where applicable
- every send path creates the authoritative `Sent` copy through core `LPE`
- `Bcc` remains protected from user search and normal shared-mailbox projections
- public autodiscover/autoconfiguration publishes only implemented and explicitly enabled endpoints
- local unit/integration tests pass for the relevant crates
- live harnesses pass before enabling public client metadata in a deployment

Required local verification:

```powershell
cargo test -p lpe-jmap
cargo test -p lpe-imap
cargo test -p lpe-activesync
cargo test -p lpe-exchange
cargo test --manifest-path LPE-CT/Cargo.toml
bash LPE-CT/installation/debian-trixie/test-lpe-ct-spool-recovery.sh
```

Required live verification before public publication:

```powershell
perl tools/jmap_tester_big_three_check.pl
python tools/jmap_live_shared_delegated_check.py
python tools/ews_live_smoke_check.py
python tools/rca_outlook_connectivity_check.py --ews-readiness --allow-mutating-fixtures
python tools/rca_outlook_connectivity_check.py --outlook-rca-readiness --allow-mutating-fixtures
```

The live commands require a running `LPE` deployment, configured test accounts, and the environment variables documented by each tool. They are release/publication gates, not compile-time gates. The project-owned `rca_outlook_connectivity_check.py` command is local scripted harness evidence; it does not replace Microsoft Remote Connectivity Analyzer evidence or real Outlook 2016 / 2019 cached-mode profile evidence.
