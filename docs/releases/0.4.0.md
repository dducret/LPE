# LPE 0.4.0

`0.4.0` is a fresh-database release.

## Upgrade Policy

- The core `LPE` database must be empty before applying the `0.4.0-sql-v2`
  schema with `installation/debian-trixie/init-schema.sh`.
- `update-lpe.sh` performs a read-only schema-version check, then rebuilds and
  redeploys code and web assets only; it does not run SQL compatibility patches.
- Use `LPE_RESET_SCHEMA=true` with `init-schema.sh` only for an intentional
  destructive reset of a disposable or explicitly rebuilt deployment.

## Operator Notes

- After initialization, `check-lpe.sh` verifies that the installed database
  reports schema version `0.4.0-sql-v2`.
